## Features

- **10,000 unique bilingual quotes** — every quote is a distinct combination of 7 pools in both English and Arabic, guaranteed no repeats
- **Display mode toggle** — switch between **Bilingual** (EN+AR together), **English only**, or **Arabic only**
- **Surprise Me** — random quote hero section for instant inspiration
- **Real-time search** — filter all 10,000 quotes instantly by keyword (matches both languages)
- **Pagination** — 18 quotes per page (556 pages total)
- **Concept tags** — each card shows its AI concept (Alignment, Reward Hacking, etc.)
- **Copy & Download** — per-quote clipboard copy or single .txt download (in selected language)

- **Dark / Light mode** — toggle with persistence via localStorage
- **Fully client-side** — no server, no build step, no database

## How It Works

### The 7 Lexical Pools (Bilingual)

| Pool | Count (per language) | Purpose | Example (EN) | Example (AR) |
|------|---------------------|---------|--------------|--------------|
| `conceptPool` / `conceptPoolAr` | 36 | AI/ML terminology | "Alignment" | "المواءمة" |
| `insightPool` / `insightPoolAr` | 18 | Verb phrase | "optimizes for density, not meaning" | "تُحسِّن للكثافة، لا للمعنى" |
| `riskPool` / `riskPoolAr` | 15 | Consequence | "silent failure modes multiply" | "أنماط الفشل الصامتة تتضاعف" |
| `killPool` / `killPoolAr` | 12 | Final punchline | "interpretability is not optional" | "قابلية التفسير ليست خياراً" |
| `openingPool` / `openingPoolAr` | 8 | Opening variant | "does not finalize", "is a mirror, not a map" | "لا يحسم"، "هو مرآة لا خارطة" |
| `transitionPool` / `transitionPoolAr` | 6 | Risk intro | "Left unchecked,", "The paradox:" | "ولو ترك بدون رقابة،"، "المفارقة:" |
| `tipPool` / `tipPoolAr` | 10 | Actionable tip | "Audit your tools. Question your metrics." | "دقِّق أدواتك. تحَدَّ مقاييسك." |

Total possible combinations per language: **36 x 18 x 15 x 12 x 8 x 6 x 10 = 55,987,200**

### Quote Structure (Bilingual)

Each quote is generated in both English and Arabic using parallel lexical pools with matching random indices, then displayed together:

```english
{Concept} {opening} — it {insight}.
{transition} {risk}.
{tip}
{kill}

© Ayman Elmasry
```

```arabic
{المفهوم} {الافتتاحية} — إنها {الرؤية}.
{الانتقال} {الخطر}.
{النصيحة}
{الخاتمة}

© أيمن المصري
```

**Bilingual display example:**
```
Alignment is not a solution — it optimizes for density, not meaning.
Left unchecked, silent failure modes multiply without warning.
Pause. Ask yourself: what am I optimizing for?
Interpretability is not optional — it is survival

─────────────────────

المواءمة ليست حلاً — إنها تُحسِّن للكثافة، لا للمعنى.
ولو ترك بدون رقابة، أنماط الفشل الصامتة تتضاعف دون إنذار.
توقَّف. اسأل نفسك: لماذا تُحسِّن؟
قابلية التفسير ليست خياراً — بل نجاة
```

### Generation Algorithm

1. Each pool index is chosen via `Math.random()`, guaranteeing uniform distribution.
2. The 7-item combination `{conceptIdx}|{openingIdx}|{insightIdx}|{transitionIdx}|{riskIdx}|{tipIdx}|{killIdx}` is checked against a `Set`.
3. If unique, English and Arabic quotes are assembled in parallel using the same indices — ensuring both versions share the identical conceptual structure.
4. If a duplicate (rare — expected ~0.89 collisions in 10,000 draws), it regenerates.
5. The loop continues until exactly 10,000 unique bilingual quote pairs are produced — no fallback needed.
6. **Display mode** lets you toggle between Bilingual (EN+AR together), English only, or Arabic only.

## Usage

### Browse Quotes
- Open `index.html` — quotes load automatically.
- Use **Prev / Next** to paginate (18 per page).
- Click **Surprise Me** for a random quote in the hero section.

### Search
- Type in the search box to filter quotes in real time.
- Matches against all quote text (case-insensitive).

### Copy a Quote
- Click the **copy icon** on any card to copy to clipboard.

### Download a Single Quote
- Click the **download icon** to save as `.txt`.

### Export Dataset
- Use the export buttons in the toolbar to download filtered or full results.
